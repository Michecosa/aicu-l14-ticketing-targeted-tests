# Materiali studenti - Lezione 14

## Cosa fai

Scrivi o completi un test reale su `urgencyLabel`.

Prima scegli:

```txt
rischio -> livello -> file -> comando
```

## File

- `consegna.md` - istruzioni.
- `template/test-notes.md` - nota breve su comando e output.
- `template/test-review-checklist.md` - checklist opzionale per rivedere un test generato da AI.

## Output atteso

```txt
file toccato
test
comando eseguito
output essenziale
rischio protetto
```

Non serve un report lungo.
Serve un test che gira e che sai spiegare.
